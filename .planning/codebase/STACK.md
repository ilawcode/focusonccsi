# Technology Stack

## Core
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **UI Library**: React 19
- **Database**: MongoDB (Mongoose)
- **Authentication**: NextAuth.js 4

## Styling
- **CSS Framework**: Bootstrap 5
- **Custom Styles**: Vanilla CSS (globals.css)

## Utilities
- **Encryption**: bcryptjs (for passwords), custom AES encryption (for Jira tokens)
- **Type Definitions**: TypeScript @types for Node, React, and MongoDB

## Environment
- Node.js (>=20)
- Environment variables for Jira URL and Database connection
